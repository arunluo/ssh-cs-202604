# 更新日志

所有重要的版本更新都会记录在此文件。

## [1.0.0] - 2026-05-04

### Added
- 首页水滴动画，进度实时显示
- 记一杯功能，支持选择杯量（150ml/200ml/300ml/500ml）和自定义输入
- 今日饮水数据追踪
- 连续达标天数徽章
- 记录页：今日/本周视图切换
- 趋势页：近7天/30天数据展示
- 徽章墙：成就系统
- 设置页：个人信息、目标设置、提醒配置
- 隐私政策弹窗
- 引导流程：首次使用配置

### Fixed
- 修复饮水量字符串拼接导致的计算错误
- 修复删除记录后数据不同步问题
- 修复水滴动画在空状态下的显示
- 修复趋势页排序问题（今日应显示在最前）

### Changed
- 优化波浪动画效果
- 改进 TabBar 图标设计
