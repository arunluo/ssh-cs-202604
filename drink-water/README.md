# 喝喝水小程序

一款温柔提醒、不骚扰用户的喝水小助手。

## 功能

- 💧 记录饮水 - 一键记录喝水量，支持多种饮品类型
- 📊 进度追踪 - 水滴动画实时显示进度
- ✅ 每日任务 - 早起、餐前、睡前提醒，完成得积分
- 🎯 目标设定 - 根据体重推荐每日饮水目标
- 🔔 智能提醒 - 固定间隔提醒 + 勿扰时段 + 预设模板
- 🏆 成就徽章 - 连续达标解锁成就，扩展6个新成就
- 📈 趋势分析 - 近7天/30天数据，含周报分析
- 💬 喝水语录 - 首页展示健康提示和鼓励语
- 📤 分享成就 - 海报分享给好友

## 技术栈

- 微信原生 API
- JavaScript
- WeUI 组件库

## 本地存储

所有数据均存储在本地（wx.setStorageSync），不上传服务器。

## 使用

1. 在微信开发者工具中导入 `src` 目录
2. 填写 AppID（或使用测试号）
3. 编译运行

## 目录结构

```
src/
├── app.js              # 应用入口
├── app.json           # 路由配置
├── app.wxss           # 全局样式
├── pages/
│   ├── index/         # 首页 - 水滴动画、记一杯、语录
│   ├── record/        # 记录页 - 今日/本周/趋势
│   ├── tasks/         # 任务页 - 每日任务
│   ├── badges/        # 徽章墙 - 成就展示
│   ├── settings/      # 设置页 - 个人信息、提醒设置
│   └── share/         # 分享页 - 海报生成
├── components/        # 公共组件
├── utils/            # 工具函数（storage, reminder）
└── styles/images/    # TabBar 图标
```

## 隐私

本小程序所有数据均存储在您的本地设备，不会上传至任何服务器。
