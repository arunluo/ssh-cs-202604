# 更新日志

所有重要的版本更新都会记录在此文件。

## [2.0.0] - 2026-05-05

### Added
- **饮品类型选择**：支持水、咖啡、茶、果汁、其他，每种有不同补水系数
- **每日任务系统**：早起第一杯、餐前喝水、睡前一cup，完成获得积分
- **分享功能**：Canvas海报生成，支持分享给好友和保存图片
- **周报功能**：本周vs上周对比、最佳日期、达标天数、个性化建议
- **首页喝水语录**：健康提示、鼓励语、趣味语混合展示
- **提醒优化**：预设提醒模板 + 用户自定义，根据起床时间智能调整首次提醒
- **成就徽章扩展**：新增6个成就（咖啡爱好者、茶道大师、周末水怪、早起达人、百杯达人、夜猫子）

### Changed
- **记录页重构**：合并趋势功能，变为今日/本周/趋势三个Tab
- **tabBar优化**：首页、记录、任务、徽章、设置（限制5项）
- 分享海报尺寸优化（540x720）

### Fixed
- 修复tabBar最多5项限制
- 修复share页面wx.shareAppMessage API调用错误
- 删除trend目录（功能已合并到记录页）

### Technical
- 任务图标重新设计（剪贴板风格）
- 修复Canvas海报生成超时问题

## [1.0.0] - 2026-05-04

### Added
- 首页水滴动画，进度实时显示
- 记一杯功能，支持选择杯量（150ml/200ml/300ml/500ml）和自定义输入
- 今日饮水数据追踪
- 连续达标天数徽章
- 记录页：今日/本周视图切换
- 趋势页：近7天/30天数据展示
- 徽章墙：成就系统
- 设置页：个人信息、目标设置、提醒配置
- 隐私政策弹窗
- 引导流程：首次使用配置

### Fixed
- 修复饮水量字符串拼接导致的计算错误
- 修复删除记录后数据不同步问题
- 修复水滴动画在空状态下的显示
- 修复趋势页排序问题（今日应显示在最前）

### Changed
- 优化波浪动画效果
- 改进 TabBar 图标设计
