# 代码优化器

Python 代码分析和优化工具集。

## 功能

- 代码分析
- 性能分析
- 自动重构建议

## 使用方法

```bash
# 运行主程序
python main.py

# 代码分析
python analyzer.py

# 性能分析
python profiler.py

# 重构建议
python refactor.py
```

## 模块说明

| 文件 | 功能 |
|------|------|
| `main.py` | 主入口 |
| `analyzer.py` | 代码分析器 |
| `profiler.py` | 性能分析器 |
| `refactor.py` | 重构建议生成器 |
