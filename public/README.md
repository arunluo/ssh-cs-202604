# 背景图片说明

如果您有江西农业大学的实际校园照片，请按照以下步骤替换背景图片：

## 替换背景图片步骤：

1. **准备图片**：
   - 将江西农业大学校园照片保存为 JPG 或 PNG 格式
   - 建议使用高分辨率图片（至少1920x1080）
   - 推荐文件名：`jxau-campus.jpg`

2. **放置图片**：
   - 将图片文件放入此 `public` 文件夹
   - 或者放入 `src/assets` 文件夹

3. **修改代码**：
   - 打开 `src/App.tsx` 文件
   - 找到 `backgroundImage` 样式设置
   - 将 URL 替换为您的图片路径：
     ```tsx
     backgroundImage: "url('/jxau-campus.jpg')",
     ```

4. **调整透明度**：
   - 根据需要调整覆盖层的透明度
   - 当前设置为 `from-pink-300/60`（60%不透明度）
   - 可以调整为 `from-pink-300/40`（40%不透明度）以获得更好的背景显示效果

## 推荐图片规格：
- 格式：JPG 或 PNG
- 分辨率：≥ 1920x1080
- 文件大小：≤ 2MB
- 内容：江西农业大学校园风景、建筑或标志性场景

## 效果调整：
- 如果背景太亮，可以增加覆盖层的不透明度
- 如果背景太暗，可以降低覆盖层的不透明度
- 可以调整 `backdrop-blur-[2px]` 来改变模糊效果
